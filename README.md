# Multi-year overwintering fires search algorithm and analyses

This repository contains code and documentation for the multi-year overwintering fire search algorithm and analyses presented in Wilkinson et al. The detection algorithm is is split into 5 steps and can be found in `detection_algorithm/` and a description of the algorithim is found in `detection_algorithm/description.md`. The datasets used are described in`supplmentary/datasets_used.md`.  The overwintering detection criteria are detailed in `supplementary/overwintering_criteria.md`.

All software requirements for python are outlined in `pyproject.toml` and requirements for R are outlined in `uvr.toml`

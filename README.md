# Multi-year overwintering fires search algorithm and analyses

This repository contains code and documentation for the multi-year overwintering fire search algorithm and analyses presented in Wilkinson et al. The main script files are `analysis_sc.py` for the spatial and temporal proximity criteria and `analysis_stp.py` for the spatial continuity criteria. The overwintering detection criteria are detailed in `supplementary/overwintering_criteria.md`.

All software requirements for python are outlined in `pyproject.toml` and requirements for R are outlined in `uvr.toml`
